# BUG-002

## Title
Form Accepts Empty Input

## Module
Contact / Signup Form

## Environment
- Browser: Chrome
- OS: Windows 11

## Severity
High

## Priority
High

## Preconditions
User is on the form page.

## Steps to Reproduce

1. Open the form.
2. Leave all fields empty.
3. Click Submit.

## Actual Result

Form is submitted without validation.

## Expected Result

Validation message should appear.

## Status

Open
