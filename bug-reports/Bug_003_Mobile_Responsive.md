# BUG-003

## Title
Button Overlaps Text on Mobile Screen

## Module
Homepage CTA Section

## Environment
- Browser: Chrome Mobile
- Device: Android

## Severity
Medium

## Priority
Medium

## Preconditions
Homepage is opened in mobile view.

## Steps to Reproduce

1. Open homepage.
2. Switch to mobile view.
3. Scroll to CTA section.

## Actual Result

Button overlaps page text.

## Expected Result

Button should display below the text without overlap.

Open
